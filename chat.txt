Esses arquivos correspondem a um programa que usa a API do OpenAI para enviar mensagens escritas pelo usuário para um modelo de linguagem natural (no caso, o modelo GPT-3.5-turbo) e retorna uma resposta também escrita em linguagem natural. 

Para isso, é necessário passar um prompt (um texto que dá o contexto ou a instrução para a resposta), que é passado pelo usuário. O programa também conta com a função de scrap_sistem, que lê os arquivos com extensão especificada (por padrão, .py) de diretórios específicos e retorna o código dentro de blocos de código. 

As mensagens enviadas e recebidas são apresentadas em formato de diálogo e armazenadas em arquivo .txt. O programa admite a passagem de flags para especificar os diretórios que devem ser lidos. Além disso, são fornecidos outros auxiliares como um getToken para obter o token de acesso à API do OpenAI e um setToken para obter e salvar o token.
