# IAp

O pacote iap é uma ferramenta para integrar o OpenAI GPT ao seu computador, permitindo que você faça consultas e obtenha respostas assertivas em texto.

# Instalação

Para instalar o pacote, utilize o `pip`:

```
pip install iap
```

# Configurando a API Key

Antes de utilizar o pacote, é preciso obter uma API Key na [OpenAI](https://beta.openai.com/), e configurar a variável de ambiente `OPENAI_API_KEY` com o valor da sua chave.

Também é possível armazenar a API Key em um arquivo de texto chamado `iap_token.txt`, que deve ser colocado em um nível acima do diretório que contém o código fonte do projeto.

# Utilização

Para utilizar o pacote, basta chamar o script principal `iap` e inserir o prompt no formato de uma pergunta. Por exemplo:

```
iap "Qual é a capital do Brasil?"
```

O programa então enviará a pergunta para o OpenAI GPT, e retornará a resposta gerada. O conteúdo da resposta será salvo em um arquivo chamado `chat.txt`. Além disso, o código gerado pela resposta será salvo em um arquivo especificado pelo usuário no segundo parâmetro. Se nenhum segundo parâmetro for fornecido, o arquivo padrão será `chat.py`.

Para facilitar a utilização do pacote, é possível também criar scripts para automatizar perguntas e aquisições de respostas, e ainda utilizá-las ao longo de outros programas.

## Scripts de ajuda

O pacote também fornece dois scripts de ajuda para gerenciamento do arquivo `iap_token.txt`:

* `setToken`: configura a API Key no arquivo `iap_token.txt`.
* `getToken`: retorna o conteúdo do arquivo `iap_token.txt`, que deve conter a API Key.

Ambos os scripts podem ser chamados diretamente pelo terminal pelo seu nome.
